Our GitHub repository has a wealth of examples to explore and try out. You can check out our [examples folder](https://github.com/marcusschiesser/vectorstores/tree/main/examples) to see them all at once, or browse the pages in this section for some selected highlights.

## Use examples locally

It may be useful to check out all the examples at once so you can try them out locally. To do this into a folder called `my-new-project`, run these commands:

```bash npm2yarn
npx degit marcusschiesser/vectorstores/examples my-new-project
cd my-new-project
npm i
```

Then you can run any example in the folder with `tsx`, e.g.:

```bash npm2yarn
export OPENAI_API_KEY=your-api-key
npx tsx ./retrieval/starter.ts
```
